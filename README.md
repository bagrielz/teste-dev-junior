## 🗂 Como baixar o projeto

```bash
    # Clonar o repositório
    $ git clone https://github.com/bagrielz/to-do-list

    # Entrar no diretório
    $ cd teste-dev-junior

    # Instalar as dependências
    $ npm install

    # Iniciar o projeto
    $ npm run dev
```
